CSS中absolute的特性和应用

CSS

  以前有稍微的看过一下HTML和CSS，不过也只是了解的程度，想自己写些布局还是有些困难。因为有时候可能还是会用到一些，所以再稍微深入了解一下，至少是可以做些简单的布局吧。这里就做一些简单的记录。这次先写一下absolute。

  absolute和浮动有相同的两大特性：破坏性和包裹性。破坏性：子元素设置absolute，容器高度坍缩。包裹性：容器设置absolute，容器大小刚好包裹子元素，前提是容器没有设置宽高。下面展示一下，什么是包裹性和破坏性。因为结构比较简单，这里只贴一下CSS。

  这两个特性是在使用absolute时时刻需要注意的地方。当然这并不一定是负面影响，主要还是要结合实际的应用场合。

  absolute还有一些巧妙的应用，第一个是位置跟随。位置跟随的意思是，当一个元素的position设置为absolute，并且没有设置left、right、top和bottom时，元素会保持在原来位置，并且处于浮动状态。这里一个重点是没有设置left、right、top和bottom，如果设置了，元素会根据最近的设置有positon:relative/absolute/fixed/sticky的元素设置位置。这种情况下，元素会保持在原来位置，并且不占据空间，这时候可以通过设置margin值来调整位置。这里看两个例子[1] [2]。

  对于第一个例子，使用absolute之后，因为vip图标本身不占据空间，并且位置不变（这里有个技巧，具体请看参考链接），所以跟在图片后面，这时候设置一下margin值，就可以轻松达到覆盖图片左上角的效果，即便图片大小有变化，依然可以保证vip图标的位置。

  对于第二个例子，这是一个图标对齐和文字的对齐，也可通过设置absolute来做到对齐效果。

  还有一个应用是设置left、right、top和bottom属性达到拉伸或填充的效果。例如设置left:0和right:0，结果就是左右拉伸，效果和width:100%是一样的。这个比较好理解，就不多说了。

  absolute的特性大致就这些，巧妙的应用能达到很好的效果。这里感谢张鑫旭老师CSS深入理解之absolute的视频，更详细的说明可以看看老师的视频。不过张老师比较有个性，风格可能不是大家都能适应。
